How to use in Tesla browser

  1. Start the xClaw backend on a machine with network access.
  2. In your Tesla, open the browser and navigate to <backend-url>/dashboard/.
  3. Bookmark the page for quick access.
  4. Ask the AI about your vehicle, charging, climate, etc.
